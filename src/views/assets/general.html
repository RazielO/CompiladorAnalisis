Fases de analisis de un compilador

Hecho por: Oscar Raziel Andrade Ramírez

Materia: Lenguajes y automatas 2

Semestre: Agosto - Diciembre 2019

Contenidos principales:

  • Analisis lexico
  • Analisis sintactico
  • Analisis semantico

Herramientas usadas:

  • Java. Lenguaje de programación en el que fue desarrollado este proyecto.
  • JavaFX. Libreria para el desarrollo de la interfaz de usuario.
  • HTML. Lenguaje de marcado usado para mostrar la información de cada análisis (y esta página).
  • CSS. Lenguaje para dar formato a los documentos HTML.
  • Librerias externas para Java
    • RichTextFX. Libreria para crear un area de texto especial para codigo.

Otras herramientas usadas que no se relacionan directo con el desarrollo:

  • Git. Software usado para tener un control de versiones de este proyecto.
  • Github. Sitio web usado para alojar este proyecto.
  • Markdown. Lenguaje de marcado para generar la documentación e investigación necesaria.
  • Boostnote y Typora. Software usado para editar archivos tipo markdown.
